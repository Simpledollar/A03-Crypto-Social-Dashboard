# 🕹️ A03 Crypto Social Dashboard

Một dashboard retro neon để theo dõi giá crypto realtime và thảo luận ngay trong **Social Feed**.

## ✨ Tính năng
- Chart realtime cho BTC, ETH, SOL, BNB, DOGE (dữ liệu từ Binance API).
- Social Feed arcade-style: nhập tin nhắn, hiển thị như console chat.
- Alert giá hiển thị trong feed (không pop-up, không treo chart).
- Giao diện neon xanh tím phong cách retro.

## 🚀 Cách chạy
```bash
pip install -r requirements.txt
python dashboard.py
